# Smoking vs Non-Smoking Image Classifier

This project uses MobileNetV2 and transfer learning to classify images as smoking or non-smoking.  
Evaluation is done using K-Fold Cross Validation (5 folds). Achieved ~96% validation accuracy.

## Features
- ✅ Transfer learning with MobileNetV2
- ✅ K-Fold training loop
- ✅ Grad-CAM visualization (optional)
- ✅ Lightweight and accurate

## Dataset
Custom dataset of labeled smoking and non-smoking images. Not included in the repo due to size.
